# SpendWise Expense Tracker

A modern, responsive expense tracker web application built with HTML5, Tailwind CSS, and vanilla JavaScript.

## Features
- **Modern UI**: Sleek glassmorphic design with dark/light mode.
- **Transaction Tracking**: Add, view, and delete income/expense transactions.
- **Data Visualization**: Interactive Doughnut chart for category breakdown using Chart.js.
- **Persistence**: All data is saved to your browser's local storage.
- **Export**: Download your transaction history as a JSON file.
- **Responsive**: Fully optimized for mobile and desktop views.

## Technologies
- HTML5 & CSS3
- Tailwind CSS (Utility-first styling)
- Vanilla JavaScript (ES6+)
- Chart.js (Visualization)
- Font Awesome (Icons)

## Getting Started
1. Clone or download the project.
2. Open `index.html` in any modern web browser.
3. Start tracking your expenses!

## Project Structure
- `index.html`: Main application interface.
- `css/style.css`: Custom animations and glassmorphism.
- `js/app.js`: Main application logic and UI orchestration.
- `js/storage.js`: LocalStorage handling.
- `js/chart.js`: Chart initialization and updates.
- `js/utils.js`: Formatting and helper functions.
